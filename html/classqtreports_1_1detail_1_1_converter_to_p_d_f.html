Подробное описание

Класс, используемый для конвертации документа в PDF формат.

Заметки
Все функции данного класса, возвращающие bool, возвращают true в случае успеха либо false, если во время выполнения произошли ошибки. При этом с помощью метода getLastError() можно получить описание последней произошедшей ошибки.

Класс, используемый для конвертации документа в PDF формат. Поддерживает установку DPI.

int qtreports::detail::ConverterToPDF::m_currentHeight
private

Значение текущей высоты кисти, рисующей на листе документа PDF.

Значение постоянно меняется в зависимости от высоты band. Например, если в отчете для band'а, который отрисовывается в данный момент, значение параметра height равняется 35, то после отрисовки данного band'а, кисть переместится вниз на 35 единиц.
